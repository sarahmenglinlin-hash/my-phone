## ✨ 核心价值提示

记住，小手机的**核心价值**是：

> **Tier 1**: 人物设定的准确执行，确保AI人物不会OOC（出戏）

所有的功能、技术决策和设计都应该围绕这个核心价值进行。简洁的UI、高效的RAG检索、精心设计的CoT预设——这些都是为了保证人物的一致性和可信度。
